## Push prisma database:

npx prisma db push

## Regenerate prisma data:

npx prisma generate

## Run project with backend server:

Vercel dev

## Clean the Prisma database and generate it anew based on your schema:

npx prisma migrate reset

## Technology Stack

- **Frontend:** React, Material UI, Tailwind CSS, Vite, TypeScript
- **Backend:** TypeScript (API routes), Node.js
- **API:** REST (custom endpoints in `api/`)
- **Database:** PostgreSQL (via Prisma ORM)
- **Testing:** Vitest, jsdom, React Testing Library
